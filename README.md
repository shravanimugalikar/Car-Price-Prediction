# **Car-Price-Prediction**
This project is an end-to-end Car Price Prediction system built using Machine Learning.  It predicts the resale price of used cars based on features such as car name, company, year, kilometers driven, and fuel type.  

**Project Overview**
- Problem Type: Regression
- Algorithm Used: Linear Regression
- Frameworks & Libraries: Python, Pandas, NumPy, Scikit-learn, Streamlit
- Deployment: Streamlit App 

**Dataset**
- Source: Quikr (used car listings)
- Format: CSV
- Raw File: quikr_car.csv
- Cleaned File: Cleaned Car.csv

**Key Features:**
- name – Car model name
- company – Manufacturer
- year – Manufacturing year
- kms_driven – Kilometers driven
- fuel_type – Fuel type
- Price – Selling price (target variable)
